# Anti-Harmony-Module

# 这是什么?
这是安装[Anti-Harmony](https://github.com/BlueArchiveCN/Anti-Harmony/)项目脚本的Kernel SU/Magisk模块。

# 我该怎么使用?
Release里下载zip文件，直接在Kernel SU/Magisk管理器中刷入即可使用。

# 失效情况

- 大版本更新
- 手动重新下载资源

失效后请**重新安装**
